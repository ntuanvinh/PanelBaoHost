**Panel BaoHost** is a modern Android application designed for hosting service management. Developed using **Kotlin** and assisted by **Gemini AI**, it focuses on providing a clean user experience and efficient performance.

## ✨ Key Features

- 🚀 **Intuitive Dashboard:** Easily manage and monitor your BaoHost services on the go.
- 🤖 **AI-Assisted Development:** Optimized code structures and logic powered by Gemini AI.
- 🛡️ **Verified Safe:** Scanned and verified by VirusTotal for your security.

## 🛠 Getting Started

### For Users (APK Installation)
1. Go to the [Releases](https://github.com/ntuanvinh/PanelBaoHost/releases) page.
2. Download the latest `.apk` file.
3. Open the file on your Android device and install.

### For Developers (Build from Source)
1. Clone the repository: ```git clone https://github.com/ntuanvinh/PanelBaoHost.git```
2. Open in **Android Studio**.
3. Build and Run.
